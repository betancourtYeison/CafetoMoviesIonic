# CafetoMoviesIonic
Cafeto Test. Aplication built with ionic.

## 1. Git clone

Run `git clone https://github.com/betancourtYeison/CafetoMoviesIonic.git` to download repository

## 2. Install Packages

Run `npm install` to install all the packages for the project

## 3. Running App

Run `ionic serve`.
